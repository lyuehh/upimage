## upimage
Upload image to public CDN via cli.

## Usage

```
npm install -g upimage
```

then
```
upimage -s jd -f ~/xx.jpg
```

done.


## Available server

`['alibaba', 'jd', 'netease', 'qihoo', 'smms', 'sogou', 'xitu']`

## Warning
Personal use only, no guarantee of availability.
